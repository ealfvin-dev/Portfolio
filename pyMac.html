PyMac

  • Data reduction software in Python to process precision measurement data.
  • The software uses a least squares best fit matrix solution to calculate results from measurement data.
  • An F and T test are performed to evaluate the standard deviation (residuals) of the best fit, and the calculated results of a known weight in the design against a reference value, respectively.
  • Front end is written in Kivy using an MVC structure.
  • User interface incorperates an interactive text editor with custom linting features.
